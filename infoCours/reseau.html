About this Course

A computer network (in English: data communication network or DCN) is a set of equipment linked together to exchange information. By analogy with a net (a network is a "small net", i.e. a small net1), the end of a connection is called a node, which can be an intersection of several connections or equipment (a computer, a router, a hub, a switch). Regardless of the underlying technology, we generally take a matrix view of what a network is. Horizontally, a network is a stratum of three layers: infrastructure, control and command functions, services provided to the user. Vertically, a geographical division is often used: local network, access network and interconnection network.
