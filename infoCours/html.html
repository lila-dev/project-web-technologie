About this Course

HTML stands for “HyperText Markup Language” which can be translated as “hypertext markup language”. It is used to create and represent the content of a web page and its structure. Other technologies are used with HTML to describe the presentation of a page (CSS) and/or its interactive features (JavaScript). "Hypertext" refers to the links that link web pages together, whether within the same website or between different websites. Links are a fundamental aspect of the web. They are the ones who form this “web” (this word is translated as web in English). By downloading content from the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.
